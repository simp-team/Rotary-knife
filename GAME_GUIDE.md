# 遊戲玩法說明書 (Game Guide)

## 1. 遊戲目標
這是一個多人亂鬥生存遊戲。玩家需要操控角色在地圖上收集刀子、擊殺敵人，並不斷進化自己的階級，最終成為場上最強的霸主。

## 2. 核心玩法
*   **移動**：滑鼠控制角色移動方向。
*   **收集**：地圖上會隨機生成刀子（Pickups），靠近即可拾取，增加身邊旋轉的飛刀數量（最大 100 把）。
*   **戰鬥機制**：
    *   **刀環碰撞 (Grinding)**：當雙方的飛刀圈接觸時，會產生輕微的物理阻力。此時雙方的刀子會互相抵銷（1換1消耗）。
    *   **階級壓制**：高階級的角色在碰撞時，有較高機率摧毀低階級的刀子而不損失自己的刀子。
    *   **擊殺判定 (Fatal Hit)**：
        *   這是遊戲的核心殺招。只要你的**飛刀圈範圍**接觸到敵人的**身體本體**，敵人就會**瞬間死亡**。
        *   無視敵人剩餘刀數，只要突破防禦圈摸到本體就是必殺。
        *   若雙方同時摸到對方本體，攻擊範圍（刀數）較多的一方獲勝。

## 3. 升級系統 (Evolution)
當你收集滿 **100 把飛刀** 時，畫面會出現「Upgrade」按鈕。點擊後，你的階級會提升，飛刀數量重置，但顏色改變且判定變強。

**階級進化路線：**
1.  🟡 **Yellow (黃色)** - 初始階級
2.  🔵 **Blue (淺藍)**
3.  🔵 **Dark Blue (深藍)**
4.  🟣 **Purple (紫色)**
5.  🌸 **Pink (粉紅)**
6.  🔴 **Dark Red (深紅)** - 最終階級

## 4. 獲勝條件 (Victory)
要贏得遊戲，你需要完成究極進化：
1.  一路升級達到最高階級 **Dark Red (深紅)**。
2.  在深紅階級下，再次收集滿 **100 把飛刀**。
3.  達成後，系統將判定為 **VICTORY (勝利)**。

## 5. 失敗條件 (Defeat)
*   被敵人的飛刀圈接觸到身體中心。
*   死亡後會掉落部分刀子，遊戲結束並返回主選單。
